Website para Artista Multimídia: Faya Kayano

Projeto: Site Biográfico para Artista Musical Desenvolvi um website imersivo para o artista Fayakayano, combinando design contemporâneo com interatividade para refletir sua identidade artística multicultural. O projeto destacou-se por: Design Visual Dinâmico: Implementei animações fluidas com GSAP, incluindo um carrossel infinito interativo que responde a gestos do usuário (arrastar/inércia), reforçando a energia do artista. Experiência Multimídia: Integrei conteúdos de plataformas como YouTube e Spotify diretamente no layout, com seções dedicadas a vídeos e galeria de imagens responsivas. Técnicas Avançadas: Utilizei CSS moderno (gradientes animados, efeitos hover) e JavaScript para criar transições suaves entre seções, garantindo performance em dispositivos móveis.

Site Dra. Nicole Raposo

Esse projeto apresenta uma plataforma profissional para naturopatia e massagens terapêuticas, desenvolvida com HTML, CSS e JavaScript. Com design responsivo em tons de verde e azul, o site oferece navegação intuitiva entre serviços detalhados (naturopatia, reflexologia, acupuntura auricular, massagens terapêuticas e mais), depoimentos de clientes em carrossel interativo, informações sobre seu livro e área de contato. Os recursos técnicos incluem animações com ScrollReveal, botão "voltar ao topo", menu hambúrguer para dispositivos móveis e alertas personalizados, proporcionando uma experiência de usuário fluida e esteticamente alinhada à área de saúde natural.
