Projetos

2024

Tramonto Decor

Desenvolvi um portfólio online para a Tramonto Decor utilizando as linguagens HTML, CSS e JavaScript. O site apresenta de forma clara e intuitiva a variedade de produtos oferecidos pela empresa, como persianas, cortinas e toldos. A interface interativa permite aos visitantes navegar facilmente pelo catálogo, visualizando imagens de alta qualidade e informações detalhadas sobre cada produto.

Kula Yoga

Este projeto foi desenvolvido com o objetivo de proporcionar uma experiência digital imersiva, onde os praticantes podem encontrar tudo o que precisam para aprofundar sua prática. Através de um design intuitivo e conteúdo relevante, o site oferece informações sobre aulas, professores, benefícios do yoga e muito mais, convidando os visitantes a embarcarem em uma jornada de bem-estar.
